# Nutrition Assistant

## Project Description
Nutrition Assistant is a MERN Stack web application that helps nutritionists manage clients, create personalized meal plans, and track client progress. The system provides a simple and secure platform for maintaining nutrition-related records.

## Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Bootstrap
- Axios

## Features
- User Registration
- User Login
- Dashboard
- Client Management
- Meal Plan Management
- Progress Tracking

## Project Structure

nutrition-assistant/

## Installation

### Backend
```bash
cd server
npm install
node app.js
```

### Frontend
```bash
cd client
npm install
npm start
```
